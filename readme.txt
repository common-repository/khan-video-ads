Insert Ads over Youtube and Vimeo videos.

== Description ==
This plugin display ads over video.Ads can be Google Ads, Html5 Video ad, Images Ads, text Ads, custom ad with time and skip time options. 
Its works both in iframe and pretty embed youtube vimeo ads.Easy to use, Display option available for custom post or pages.
Its Also work with Plugin (Simple YouTube Embed) by Author: naa986.

Check Demo Here https://todaymsg.com/
== Installation ==
Upload the plugin
Go to your admin page Settings menu > Khan Video Ads
Here you can easily understand all things.
